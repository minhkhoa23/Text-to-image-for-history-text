Trong thư mục này của chúng em bao gồm:
-Thư mục image evaluation: chứa các ảnh được sinh ra từ model để sử dụng cho việc đánh giá
-Thư mục qwen: chứa các file chương trình chạy sinh ảnh từ model qwen có đánh giá CLIP, vì model này quá nặng và chạy lâu nên tụi em chia ra mỗi file chạy tạo một ảnh
-FFusionXL-BASE.ipynb: file chương trình chạy sinh ảnh từ model FFusionXL-BASE có đánh giá CLIP
-image-DPG-evaluation.ipynb: file chương trình chạy đánh giá DPG từ ảnh sinh của các model
-SD-xl-base-1.txt: chứa link file chương trình chạy sinh ảnh từ model SD-XL có đánh giá CLIP
-stable-diffusion-v1-5.ipynb: file chương trình chạy sinh ảnh từ model SD-1.5 có đánh giá CLIP
